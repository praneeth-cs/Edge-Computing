# Adaptive Traffic Intersection Simulator

A traffic simulation using Pygame featuring adaptive traffic lights, vehicle spawning, lane management, safe following distances, and turning behaviors (left, right, straight, U-turn).

## Features
- Adaptive signal control based on lane congestion
- Four-way intersection simulation
- Vehicle queue management
- Bezier-curve turning paths
- Real-time visualization

## Installation
pip install -r requirements.txt

## Run
python traffic_simulation.py

# Sensor Monitoring Dashboard

## Overview
A Streamlit-based application that simulates real-time sensor readings and monitors system behavior using adaptive thresholds, deviation analysis, alarm detection, and cloud upload decision logic.

## Features
- Real-time sensor simulation
- Adaptive threshold calculation
- Deviation monitoring
- Consecutive threshold breach detection
- Alarm triggering
- Simulated cloud upload filtering
- Live charts and metrics

## Workflow
1. Generate sensor readings
2. Calculate adaptive threshold
3. Analyze deviation
4. Monitor threshold breaches
5. Trigger alarms
6. Simulate cloud upload decisions
7. Visualize results in dashboard

## Technologies
- Python
- Streamlit
- Pandas
- NumPy

## Installation
pip install -r requirements.txt

## Run
streamlit run sensor_dashboard.py
